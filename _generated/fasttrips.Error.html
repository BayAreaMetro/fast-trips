fasttrips.Error¶

This file will define the various exceptions that can be raised in the course of running Fast-Trips.

Exceptions

ConfigurationError(filename, msg) Exception raised for errors in configuration.
DemandInputErorr(filename, msg) Exception raised for errors in the demand input.
Error Base class for exceptions in fast-trips.
NetworkInputError(filename, msg) Exception raised for errors in the network input.
NotImplementedError(msg) Exception raised for something expected but not implemented.
UnexpectedError(msg) Exception raised for something unexpected.
